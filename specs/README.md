# Specs

このディレクトリは、このプロジェクト（自前 Git リモート + 仕様/ルール可視化 Web アプリ）の仕様書を格納します。

## 仕様書一覧

- `overview.md`: 目的・スコープ・非ゴール
- `git-service.md`: Git リモート（HTTPS Smart HTTP / bare repo / hook）
- `auth-pat.md`: 認証（NextAuth）と PAT 発行・利用（push 認可）
- `repository-conventions.md`: リポジトリ内の必須構造（`specs/`, `rules/` 等）
- `web-ui.md`: Web 画面・体験
- `analytics.md`: 解析（ルール割合、テストカバレッジ等）の取り込み
- `development-environment.md`: 開発環境（docker-compose、DB接続）
